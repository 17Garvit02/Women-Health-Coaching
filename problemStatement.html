Data

Example of a Event data JSON object:

{
	"id": 2,
	"height": 50,
	"weight": 42,
	"coach": {
	    "id": 2,
	    "name": "John"
	}
}

Requirements

Here you are provided a events database where you have to build a Spring Boot application with two entities: `Coach` and `Customer`. Each `Customer` has an ID, height, and weight, while a `Coach` has an ID, name, and a set of customers. One coach can have multiple customers.

Create a Customer

  • Method: POST
  • Endpoint: /api/customer
  • Request Body: JSON
    • { "height": 170, "weight": 65, "coach": { id: 1 "name": "John" } }
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON
      • { "id": 1, "height": 170, "weight": 65, "coach": { id: 1 "name": "John" } }

Get All Customers

  • Method: GET
  • Endpoint: /api/customer
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON Array
      • [ { "id": 1, "height": 170, "weight": 65, "coach": { id: 1 "name": "John" } }, { "id": 2, "height": 160, "weight": 55, "coach": { id: 1 "name": "John" } } ]

Get a Customer by ID

  • Method: GET
  • Endpoint: /api/customers/{id}
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON
      • { "id": 1, "height": 170, "weight": 65 }

Create a Coach

  • Method: POST
  • Endpoint: /api/coache
  • Request Body: JSON
    • { "name": "John Doe" }
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON
      • { "id": 1, "name": "John Doe" }

Get a Coach by ID

  • Method: GET
  • Endpoint: /api/coach/{id}
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON
      • { "id": 1, "name": "John Doe" }

Get All Coaches

  • Method: GET
  • Endpoint: /api/coach
  • Response:
    • Status Code: 200 (OK)
    • Response Body: JSON Array
      • [ { "id": 1, "name": "John Doe" }, { "id": 2, "name": "Jane Smith" } ]

Test writing In addition to implementing the REST endpoints, you are supposed to write several(at least 3) unit tests to test your implementation.

Commands

  • run:
mvn clean spring-boot:run
  • install:
mvn clean install
  • test:
mvn clean test
